# **NOTES**

Markdown cheatsheet: https://www.markdownguide.org/cheat-sheet/

## Shortcuts:

Clear terminal screen: ***Ctrl + L***

Show/hide terminal: ***Ctrl + `***

## Commands:

list files: `ls` `ls -1`



Check code difference: `git status`

Add code/file: `git add .` (the dot means all changes)

Commit code/file: `git commit -m "message here"`

Push/execute code/file: `git push`

Create folder: `mkdir projects` (make directory names projects)

Change folder: `cd projects` (move to projects folder just created)

Current directory: `pwd` (print working directory)

Back to parent folder: `cd ..`

Create file: `touch sample.py`
